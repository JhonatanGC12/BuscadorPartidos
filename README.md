# Analizador de Resultados Históricos de Fútbol — FIFA Data

Proyecto final del curso de Programación Orientada a Objetos en Java.  
Este software permite analizar resultados de partidos internacionales desde 1870 hasta la actualidad, utilizando el dataset histórico de FIFA.

---

## uncionalidades Principales
✔ Carga automática del archivo `results.csv`  
✔ Visualización detallada de todos los partidos (tabla completa)  
✔ Búsqueda de estadísticas por equipo:  
- Victorias  
- Derrotas  
- Empates  
- Goles a favor  
✔ Interfaz gráfica construida con **Java Swing**  
✔ Programación Orientada a Objetos  
✔ Proyecto estructurado en paquetes `modelo` y `vista`  
✔ Integración lista para GitHub & SonarCloud
